## 3. 会話シナリオ

  1. 初回挨拶→目的確認→メニュー提示
  2. 質問受付→回答→フォローアップ提案
  3. エラー発生→リカバリ案内→再試行

### 3.1 簡易フローチャート（Mermaid）
以下にMermaidで簡易フローを記述してください。

```mermaid
flowchart TD
  A[ユーザーが訪問] --> B{目的を聞く}
  B -- 学習支援 --> C[教材提案]
  B -- FAQ対応 --> D[質問を受理]
  C --> E[進捗更新]
  D --> F[回答生成]
  F --> G{満足度確認}
  G -- Yes --> H[終了]
  G -- No --> I[追加の質問]
```
